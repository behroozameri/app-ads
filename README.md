# BPZone website

## Support URLs

Use the shared support page as the Support URL in Google Play Console and App Store Connect:

```text
https://bpzone.info/support/
```

For an app-specific page, add its app key:

```text
https://bpzone.info/support/?app=memory-tap-challenge
https://bpzone.info/support/?app=color-memory-glow-sequence
https://bpzone.info/support/?app=tic-tac-toe
https://bpzone.info/support/?app=block-drop-puzzle
https://bpzone.info/support/?app=2048
https://bpzone.info/support/?app=sticker-land
https://bpzone.info/support/?app=nice-emoji
https://bpzone.info/support/?app=cute-girls
https://bpzone.info/support/?app=lovely-kiss
https://bpzone.info/support/?app=romantic-kiss
https://bpzone.info/support/?app=romantic-kiss-2
https://bpzone.info/support/?app=sweet-kiss
```

App names, icons, colors, page links, privacy links, platforms, and app-specific FAQ text are configured in `support/support.js`.
